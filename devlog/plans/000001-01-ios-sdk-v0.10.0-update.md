# Plan: Update iOS Examples to Use leap-sdk v0.10.0-SNAPSHOT

## Thinking

The iOS examples currently point at a local `iOS/Package.swift` with local XCFramework paths. This was a temporary dev setup. We need to switch them to the published `leap-sdk` v0.10.0-SNAPSHOT.

Key API changes from v0.9.2 to v0.10.0-SNAPSHOT:
- Package URL: `leap-ios` → `leap-sdk`
- Products split: `LeapSDK` + `LeapModelDownloader` (separate)
- `@Generatable`/`@Guide` macros now in `LeapSDKMacros` product
- `MessageCompletion.message` → `.fullMessage`
- `ChatMessageContent.audio(Data)` → `.audio(InputAudio)` (must check exact shape)
- `GenerationFinishReason` has new cases `.interrupted`, `.constraint`, `.error`

## Plan
